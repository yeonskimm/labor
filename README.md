# 사업장 노무가이드 (대구지방고용노동청)

영세사업주를 위한 노무관리 통합 웹앱입니다. "오늘의안전" 앱과 동일한 디자인 시스템을 사용합니다.

## 기능
- ⚖️ **의무사항 자가진단** — 상시근로자 수·업종 입력 → 적용 법적 의무 62개 자동 분류
- 📋 **근로계약서 작성** — 표준/단시간/건설일용/연소근로자 4종 서식, PDF 출력
- 💵 **급여 계산기** — 소정·연장·야간·주휴수당 정확 계산, 휴게시간(유급/무급) 반영

## 로컬 실행
```bash
npm install
npm run dev
```
브라우저에서 http://localhost:5173 접속.

## GitHub Pages 배포 방법

1. 이 폴더를 GitHub 저장소로 올립니다 (저장소명을 `nomu-guide`로 권장).
2. 저장소명을 다르게 쓰려면 `vite.config.js`의 `base`와 README의 주소를 그 이름으로 바꾸세요.
3. GitHub 저장소 → **Settings → Pages → Build and deployment → Source**를 **GitHub Actions**로 설정합니다.
4. `main` 브랜치에 push하면 자동으로 빌드·배포됩니다.
5. 배포 후 주소: `https://<사용자명>.github.io/nomu-guide/`
