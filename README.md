# 포켓노무사 (대구지방고용노동청)

영세사업주를 위한 노무관리 통합 웹앱입니다. "오늘의안전" 앱과 동일한 디자인 시스템(다크+골드)을 사용합니다.

## 기능

- ⚖️ **의무사항 자가진단** — 상시근로자 수·업종 입력 → 적용되는 법적 의무 62개 자동 분류
- 📋 **근로계약서 작성** — 표준 / 단시간 / 건설일용 / 연소근로자 4종 서식, 법정 필수기재사항 반영, PDF 출력
- 💵 **급여 계산기** — 소정·연장·야간·주휴수당 정확 계산, 휴게시간(유급/무급) 반영, 정기/비정기 스케줄
- 💴 **임금명세서** — 근기법 §48② 필수항목 전부 반영, 급여 계산기에서 자동 연동, PDF 출력
- 🏦 **퇴직금 계산기** — 평균임금 / 통상임금 / DB형 / DC형 4가지 방식 구분 계산

## 로컬에서 실행

```bash
npm install
npm run dev
```
브라우저에서 http://localhost:5173 접속.

## GitHub Pages 배포 방법

1. 이 폴더를 GitHub 저장소로 올립니다.
2. **저장소 이름이 `labor`가 아니면** `vite.config.js`의 `base` 값을 저장소 이름과 똑같이 바꾸세요.
   (예: 저장소명이 `nomu`면 `base: '/nomu/'`)
3. GitHub 저장소 → **Settings → Pages → Build and deployment → Source**를 **GitHub Actions**로 설정합니다.
4. `main` 브랜치에 push하면 자동으로 빌드·배포됩니다.
5. 배포 후 주소: `https://<사용자명>.github.io/<저장소명>/`

## 주의

- 본 앱은 일반 기준에 따른 참고용이며, 개별 사실관계에 따라 적용이 달라질 수 있습니다.
- 계산 결과는 세전 기준이며, 실제 금액은 4대보험·세금 공제 후 달라질 수 있습니다.
- 정확한 사항은 사업장 또는 공인노무사, 고용노동부 고객상담센터 ☎ 1350 으로 문의하세요.

## 기술 스택

- React 18 + Vite 5
- 단일 파일 컴포넌트 (`src/App.jsx`)
- 외부 의존성 없음 (네이티브 React만 사용)
